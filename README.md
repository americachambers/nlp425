# nlp425

Currently, the conversational agent can only handle the following types of sentences:

1. (Det) N V (Det) N
"Fluffy loves fish"
"The cat loves fish"
"The cat loves the fish"
"I love you"
"I love the cat"
"The cat loves me"

2. (Det) N Copula (Det) N
"Fluffy is a cat"
"The cat is a lion"

3. (Det) N Copula Adj/Adv
"Fluffy is black"
"Fluffy is fast"
